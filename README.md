# 📧 Spam Detection System using NLP

## 🔍 Overview
This project is a Spam Detection System developed using Natural Language Processing (NLP) and Machine Learning techniques. It classifies messages as Spam or Not Spam based on their content. The system also provides real-time predictions, probability scores, and visual analysis.

---

## 🚀 Features
- Real-time spam detection
- Probability (confidence score)
- Message length & word count analysis
- Keyword detection
- Top words extraction
- Multi-page Streamlit interface
- History tracking
- Bar chart & Pie chart visualization
- Download report option

---

## 🧠 Technologies Used
- Python
- NLP (Text preprocessing)
- Scikit-learn (Multinomial Naive Bayes)
- TF-IDF Vectorizer (with N-grams)
- Streamlit (Web Interface)
- Matplotlib (Visualization)

---

## 📊 Model Details
- Algorithm: Multinomial Naive Bayes
- Feature Extraction: TF-IDF with N-grams (1,2)
- Dataset: SMS Spam Collection Dataset
